# Descripción

Este programa fue desarrollado en C# y tiene como objetivo mostrar cómo funcionan dos métodos de búsqueda dentro de un arreglo de números.

Los métodos implementados son:

Búsqueda Lineal

Búsqueda Binaria

El programa muestra primero un arreglo de números ya ordenados. Luego el usuario puede ingresar un número desde el teclado y el programa buscará ese número utilizando ambos métodos para indicar si el número se encuentra dentro del arreglo y en qué posición está.
